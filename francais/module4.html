Il comprend que ce n’est pas juste une phrase. C’est un code sacré, à décrypter…

📖 Partie 2 — Leçon simple et vivante

✨ Une phrase, c’est un peu comme une équipe :

  • Le Sujet → C’est celui qui fait l’action (la personne, l’animal, la chose)
  • Le Verbe → C’est ce qui se passe, l’action, l’état
  • Le Complément → C’est à qui ? quoi ? où ? quand ? comment ?
🧠 Astuce : une phrase peut exister sans complément, mais jamais sans verbe

🌰 Exemple 1 :

"Le chat mange la souris."

  • Sujet = Le chat
  • Verbe = mange
  • Complément = la souris

🌰 Exemple 2 :

"Kylon observe les poissons dans la rivière."

  • Sujet = Kylon
  • Verbe = observe
  • Complément = les poissons dans la rivière
