🌸 MODULE 2 – FRANÇAIS : L’accord dans le groupe nominal

Titre : “Le vendeur d’objets magiques”

🎯 Objectifs

  • Identifier un groupe nominal (déterminant + nom + adjectif)
  • Comprendre le genre et le nombre
  • Apprendre à accorder les mots ensemble
  • Corriger des erreurs dans une phrase

🌀 INTRO : La petite histoire

🧙‍♂️ Il était une fois à Bangkok, dans une ruelle cachée, un vieil homme vendait des objets étranges :
des lunettes qui lisent les pensées, des gommes qui effacent les mauvais rêves, et des stylos qui écrivent tout seuls.
Il avait un cahier magique, mais ce cahier n’acceptait que les phrases bien accordées...
Le vieil homme dit à Kylon : “Tu veux mon cahier ? Alors montre-moi que tu sais accorder les groupes nominaux !”

🧠 Leçon rapide

Un groupe nominal est composé de :

  • Le déterminant : le, la, les, un, une...
  • Le nom : robe, dragon, scooter...
  • L’adjectif : rouge, magique, rapide...

Rappel : tous les mots doivent s’accorder (masculin/féminin, singulier/pluriel).
Exemples :

  • ✅ Un scooter rapide
  • ✅ Des robes rouges
  • ❌ Une dragon rouge → Un dragon rouge
