🎨 Les Adjectifs et les Adverbes

🧾 L’Adjectif

Un adjectif est un mot qui donne des précisions sur un nom. Il dit comment est une chose ou une personne.

Exemples :

  • un grand arbre
  • une voiture rouge
  • des enfants joyeux

🧠 L’adjectif s’accorde en genre (masculin/féminin) et en nombre (singulier/pluriel) avec le nom qu’il accompagne.

💨 L’Adverbe

Un adverbe est un mot qui modifie un verbe, un adjectif ou un autre adverbe. Il donne des infos sur comment, quand, où, combien.

Exemples :

  • Il court vite.
  • Elle parle doucement.
  • Il est très gentil.

🔍 Comment les reconnaître ?

Mot Type Indice
gentille Adjectif décrit un nom
rapidement Adverbe décrit un verbe
bleu Adjectif couleur d’un objet
très Adverbe modifie un adjectif

🎮 Mini-exercice

  1. Souligne les adjectifs :
    Le petit chat noir dort sur le canapé moelleux.
  2. Souligne les adverbes :
    Il joue doucement et parle lentement.
  3. Complète avec un adjectif :
    Ma sœur est très ________.
  4. Complète avec un adverbe :
    Elle danse ________.
